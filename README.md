# 🚀 Repositorio de Proyectos MLOps

Este repositorio reúne implementaciones, ejemplos y buenas prácticas relacionadas con **MLOps**, incluyendo herramientas modernas como **Airflow**, **FastAPI**, **MLflow**, y otros frameworks que permiten construir, desplegar y mantener modelos de Machine Learning de forma escalable y reproducible.

## 📑 Contenido

- **Orquestación de flujos de trabajo**  
  - Apache Airflow para pipelines de datos y ML.  
  - DAGs reproducibles y modularizados.  

- **Despliegue de modelos**  
  - FastAPI para servir modelos en producción.  
  - Endpoints RESTful documentados y versionados.  

- **Gestión del ciclo de vida de modelos**  
  - MLflow para tracking de experimentos, registro de modelos y despliegue.  
  - Integración con CI/CD para automatizar pruebas y despliegues.  

- **CI/CD y Automatización**  
  - Workflows con GitHub Actions / GitLab CI.  
  - Validación de datos y tests unitarios para asegurar calidad.  

- **Modelado y Experimentación**  
  - Ejemplos de regresión, clasificación y series temporales.  
  - Documentación clara de cada notebook y script.  

## 🛠️ Tecnologías utilizadas

- **Lenguaje principal:** Python  
- **Frameworks:** FastAPI, MLflow, Airflow, scikit-learn, PyTorch/TensorFlow  
- **Infraestructura:** Docker, WSL/Docker Compose, Kubernetes (futuro)  
- **Automatización:** GitHub Actions / GitLab CI/CD  
- **Documentación:** Markdown, README estructurados, diagramas
